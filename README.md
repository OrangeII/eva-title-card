# eva-title-card

HTML Evangelion title card generator.

## Usage

Visit https://orangeii.github.io/eva-title-card/ and append query parameters to customize the title card:

- `med1`: First medium title line (default: "NEON")
- `med2`: Second medium title line (default: "GENESIS")
- `big`: Large title text (default: "EVANGELION")
- `ep`: Episode text (default: "EPISODE:26")
- `sub`: Subtitle text (default: "世界の中心でアイを叫んだけもの")

Example URL: [`https://orangeii.github.io/eva-title-card/?big=YOUR+TITLE&ep=EPISODE+01&sub=Your+Subtitle`](https://orangeii.github.io/eva-title-card/?big=YOUR+TITLE&ep=EPISODE+01&sub=Your+Subtitle)
